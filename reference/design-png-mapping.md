# デザイン入りPNG マッピング — ビジ友（bijiyu）

## 概要

Figmaからエクスポートしたデザイン入りPNG 9枚を画面IDに紐付けた資料。
これらはワイヤーフレームPNG（既存82枚）とは異なり、**実際のデザイン（カラー・タイポグラフィ・レイアウト）** が適用された画面デザインである。

配置先: `design-assets/screens/` に `{画面ID}-design-{pc|sp}.png` の命名規則で格納。

## 各PNGから読み取れたデザイン要素

### CON-002: 募集案件一覧

**PC版 (3カラムグリッド):**
- ヘッダー: ビジ友ロゴ（紫系）、ハンバーガーメニュー、戻るボタン
- ページタイトル: 「募集案件一覧」（紫色、太字）
- 件数表示: 「全○○件」+ 検索・ソートアイコン（右寄せ）
- カードレイアウト: `grid-cols-3`、角丸、薄グレー枠
  - サムネイル画像（ビジ友ロゴプレースホルダー）
  - 締め切りバッジ: 紫背景・白文字（「締め切り：2000/00/00」）
  - タイトル: 黒太字
  - 会社名: **紫色テキスト（リンク）**
  - メタ情報: 募集職種、人数（グレーテキスト）
  - アイコン付き情報: 報酬（💰）、エリア（📍）、募集期間（📅）
  - ボタン2つ横並び:
    - 「マイリスト登録」: **紫背景・白文字（CTA）**
    - 「詳細をみる」: **紫枠・紫テキスト（outline）**

**SP版 (1カラムリスト):**
- 同一情報を1カラムカードで縦積み
- ボタンは横並び2つを維持

### CLI-006: ユーザー詳細（職人詳細）

**PC版:**
- カード内レイアウト（白背景、薄グレー枠）
- プロフィールヘッダー:
  - アバター画像（丸、左寄せ）
  - 名前 + 年齢（「山田 太郎（30歳）」）
  - バッジ: 「本人確認済み」「CCUS登録済み」（グリーン系アイコン付き）
  - ♡ボタン（右上）
- PR動画エリア: 大きなサムネイル＋再生ボタン
- アクションボタン（中央揃え横並び）:
  - 「メッセージを送る」: 紫枠（outline）
  - 「スカウトを送る」: **紫背景（CTA）**
- **基本情報**セクション:
  - テーブル形式: ラベル（紫背景帯）+ 値（白背景行）
  - 項目: 居住地、性別、対応可能エリア
- **自己紹介**セクション: テキストブロック
- **能力**セクション:
  - テーブル形式: 対応できる職種、経験年数、保有スキル、保有資格
- **空き日程**: リスト形式（カレンダーアイコン付き）
- **発注者評価**: 「また依頼したい」12/15件 + プログレスバー（紫）+ 「詳しく見る」リンク
- フッター: アクションボタン繰り返し + 「ユーザー一覧にもどる」リンク

**SP版:**
- 同一情報をモバイル幅で縦積み
- PR動画サムネイル幅100%

### CON-009: メッセージ/スカウト詳細

**PC版:**
- ヘッダー: 「○○株式会社」（左）+ ビジ友ロゴ（中央）+ メニュー（右）
- チャットUI:
  - 相手メッセージ: アバター（左）+ 吹き出し（薄グレー背景）+ タイムスタンプ
  - 自分メッセージ: 吹き出し（右寄せ、紫系薄背景）+ タイムスタンプ
  - スカウトバッジ: 紫背景「スカウト」ラベル（最初のメッセージ上部）
- **案件カード（埋め込み）**:
  - タイトル、会社名（紫リンク）、募集職種・人数、締め切り
  - 報酬、エリア、募集期間
  - 「スカウトを断る」: 紫背景ボタン
  - 「スカウトを受ける」: 紫枠ボタン
- メッセージ入力エリア（固定フッター）:
  - カメラアイコン（左）
  - テキスト入力フィールド（「メッセージ」プレースホルダー）
  - 送信ボタン（紫丸、矢印アイコン）

**SP版:**
- 同一構成、モバイル幅対応
- スカウトを断る/受けるボタンは横並び

### テンプレートパーツ（共通UIリファレンス）

**PC版・SP版:**
CON-001（マイページ）のレイアウトと複数画面で共通利用するフォームUIのデザインリファレンス。

- **マイページ上部**:
  - 「テンプレートパーツ」タイトル + 「担当者を確認する」ボタン（紫背景）
  - ユーザープロフィールカード: アバター、名前(30歳)、職種一覧、経験年数、バッジ
  - 自己紹介テキスト
  - 「仕事を探す」ナビゲーション: 募集案件一覧 / 発注者一覧 / マイリスト（右矢印付き）
  - 稼働予定カード: 「稼動予定」バッジ、案件概要、メッセージ/応募詳細ボタン

- **基本情報フォーム**（共通フォームパターン）:
  - セクションヘッダー: 「基本情報」紫背景帯
  - 画像アップロード: 2枠、「データを選択してアップロードしてください」+ 「画像を登録する」ボタン
  - テキスト入力: 会社名・氏名（必須バッジ付き）
  - セレクト: 募集職種（必須）、求める働き方
  - テキストエリア: メッセージ
  - チェックボックス: 利用SNS（X / Instagram / TikTok）
  - ラジオ/評価: 評価入力（「また依頼したいか？」「稼働予定日にちゃんと来たか？」「指示通りに動けるか？」）
  - チェックボックスグリッド: 保有スキル（型枠設置、電気工、送配電線工...）+ 「追加する」ボタン
  - フッター: 「確認する」紫背景ボタン + 「もどる」枠線ボタン

### CLI-022: 担当者一覧

**SP版のみ:**
- ページタイトル: 「担当者一覧」（紫色、中央揃え）
- 検索バー: 「キーワードで探す」+ 虫眼鏡アイコン（紫）
- 件数表示: 「検索結果：○○件」
- リスト形式:
  - 各行: 権限バッジ（左）+ 名前 + メールアドレス + 右矢印
  - 権限バッジ色:
    - 「管理責任者」: 紫背景
    - 「管理者」: 紫背景（やや薄い）
    - 「担当者」: 紫背景
  - 区切り線: 薄いグレー
- ページネーション: 「前の20件」「次の20件」
- 「担当者新規登録」: **紫グラデーション背景 + 右矢印アイコン（CTA）**
- 「もどる」: 枠線ボタン

## デザインから確認できたカラーパレット

| 用途 | 推定色 | 備考 |
|------|--------|------|
| プライマリ（CTA背景） | 紫系 `#7B2D8E` 付近 | ボタン、タイトル、バッジ |
| プライマリ（テキストリンク） | 紫赤系 `#C02060` 付近 | 会社名リンク |
| 背景 | 白 `#FFFFFF` | ページ・カード背景 |
| テキスト | 黒 `#1A1A1A` 付近 | 本文 |
| ミューテッド | グレー `#666666` 付近 | 補足情報 |
| ボーダー | 薄グレー `#E5E5E5` 付近 | カード枠、区切り線 |
| バッジ背景 | 紫系 | 締め切り、権限、スカウト |
| 成功/確認済み | グリーン系 | 本人確認済み、CCUS登録済みバッジ |

※ 正確なカラーコードはFigma/CSSデータで確定する必要あり
