- [x] 3.3 (P) 評価集計ヘルパの新規作成と単体テスト
  - リアルタイム SQL 集計（denormalize なし）で、単一受注者の総合評価サマリー・単一受注者の7項目別サマリー・複数受注者の総合評価サマリー（bulk）の3パターンを提供する

  - 任意項目の平均は「評価あり件のみ」で算出する（NULL は分母に含めない）。0件の受注者は ★平均 null / 件数 0 として返す
  - Vitest で集計ロジック（平均値の小数処理、NULL 除外、0件時の null/0）を検証する
  - 前提: 再生成済みの型（タスク 2.2）に依存。タスク 3.1/3.2/4.x とはファイル独立で並列可
